# 数据存储

## 文件清单

| 文件 | 类型 | 用途 |
|------|------|------|
| `config.json` | 配置 | AI 提供商、城市、天气代码、缓存设置、推送地址 |
| `dishes.json` | 菜品库 | 所有可用菜品及推荐理由 |
| `cache.json` | 缓存 | 推荐历史记录，用于短期不重复 |

## config.json

```json
{
  "ai": {
    "provider": "longcat",
    "base_url": "https://api.longcat.chat/openai",
    "api_key": "sk-xxx",
    "model": "LongCat-Flash-Chat-2602-Exp"
  },
  "city": "合肥",
  "weather": {
    "provider": "itboy",
    "city_code": "101220101"
  },
  "cache": {
    "no_repeat_days": 7
  },
  "push": {
    "push_url": "https://push.showdoc.com.cn/server/api/push/{token}"
  }
}
```

## dishes.json

菜品数组，每个元素包含：
- `name` — 菜名
- `reason` — 推荐理由（含时令、天气、地域饮食分析）

按 `name.strip().lower()` 去重，同名菜品保留第一条，新推荐可更新 reason。

## cache.json

```json
{
  "records": [
    {"name": "麻婆豆腐", "date": "2026-04-30", "meal": "午餐"},
    {"name": "番茄蛋汤", "date": "2026-04-30", "meal": "午餐"}
  ]
}
```

每条记录包含菜名、推荐日期和餐段。`get_excluded_dishes(days)` 根据日期过滤 N 天内记录，组装到 prompt 中让 AI 避开。
