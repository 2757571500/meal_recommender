# 需求变更：用户画像与菜品元数据

## 变更原因

原有推荐只考虑用户所在地，未考虑常驻地区口味偏好、个人口味（偏咸/偏甜等）、饮食限制等，推荐不够个性化。

## 变更内容

### 新增文件

| 文件 | 说明 |
|------|------|
| `profile.json` | 用户画像，含常驻地、口味、菜系偏好、饮食类型、忌口食材、厨艺水平、可接受烹饪时长、就餐人数 |
| `enums.json` | 枚举值配置，支持不改代码增删可选项 |

### 修改文件

| 文件 | 变更 |
|------|------|
| `config.json` | 删除 `city` 字段（迁移至 profile.hometown） |
| `dishes.json` | 清空后重建，每条菜品增加 cuisine、taste、difficulty、prep_time、ingredients、diet_type、dietary_tags 元数据字段 |
| `recommender.py` | 新增 `filter_dishes()` 硬过滤函数；`recommend()` 接收 profile；`_build_prompt()` 接收过滤后菜品和偏好信息 |
| `update_library.py` | 发现新菜时按新元数据格式生成 |
| `dish_manager.py` | 写操作保留新字段 |
| `config.py` | 新增 `load_profile()`、`load_enums()` |

## 设计方案

### 方案选择

方案 B：**本地硬过滤 + Prompt 推荐**

- 硬约束（diet_type 不匹配、超时长、超难度）在本地执行过滤
- 软偏好（口味、菜系、忌口食材）通过 prompt 告知 AI 权衡
- 理由：约束不依赖 AI 自觉性，结果更稳定

### 过滤规则

#### diet_type 匹配规则

用户 `diet_type` 为"无限制"时不过滤，保留全部菜品。
用户为特定类型时，保留"匹配类型 + 无限制"的菜品：

| 用户 diet_type | 保留的菜品 diet_type |
|---|---|
| `无限制` | 全部 |
| `素食` | `素食` + `无限制` |
| `清真` | `清真` + `无限制` |
| `纯素` | `纯素` + `无限制` |

```
if profile.diet_type == "无限制":
    pass  # 不过滤，保留全部
else:
    keep_types = {profile.diet_type, "无限制"}
```

#### 难度过滤

菜品 `difficulty` 的等级不高于用户 `skill_level`（新手 < 普通 < 熟练）。同级保留。

#### 耗时过滤

菜品 `prep_time` ≤ 用户 `max_cook_time`。

详见 `资料库/项目架构/数据存储.md` 新增章节。

### 菜品发现格式

菜品发现不再使用旧的 markdown 文本解析，AI 直接返回 **JSON 数组**：

```
请严格按以下 JSON 格式返回，不要包含其他内容：
[
  {
    "name": "菜名",
    "reason": "推荐理由",
    "cuisine": "从枚举中取值",
    "taste": ["从枚举中取值"],
    ...
  }
]
```

脚本收到后逐个校验字段值是否在 enums.json 范围内，超范围的直接丢弃并打印告警，不阻塞其他有效菜品入库。

### 首跑空库处理

`dishes.json` 清空后，`main.py` 检测到菜品库为空时打印提示并跳过推荐，不推送失败通知（避免误导）。用户需手动跑一次 `update_library.py` 或等待每周一自动调度填充。

## Prompt 变化

推荐 prompt 新增两地信息：

```
当前天气城市：北京（晴 12~22°C）
用户常驻地：合肥（口味偏好基于徽菜方向）
就餐人数：2人
口味偏好：偏辣、偏咸
菜系偏好：川菜、徽菜
忌口食材：羊肉、内脏
厨艺水平：普通
每餐可接受烹饪时间：30分钟
```

## 影响范围

- 推荐逻辑：推荐前新增过滤步骤，prompt 新增两地信息和就餐人数
- 菜品发现：AI 返回 JSON 数组，脚本校验枚举值合法性
- 推送：标题城市来源由 config.city 改为 profile.hometown
- 配置：用户首次使用需准备 profile.json、enums.json，否则推荐无法进行
- 首跑：清空 dishes.json 后需手动运行 update_library.py 填充菜品库
